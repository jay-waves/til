## 图数据结构

### 邻接表

Adjacency List. 每个顶点维护 `vector<pair>`, 适合稀疏图. 

```cpp
// i ---(w)--> j
vector<pair<j, w>>
```

### 邻接矩阵

Adjacency Matrix: `m[V][V]`, 适合稠密图 (边接近 $V^{2}$). `m[i][j]` 表示从节点 `i -> j` 的边, 无权重则存储 `INF`.
