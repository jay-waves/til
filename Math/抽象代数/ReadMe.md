
#### 数学结构

https://en.wikipedia.org/wiki/Mathematical_structure

集合上的结构指集合上附加的, 赋予集合特殊意义的数学对象. 

- 有序 (order)
- 代数
- 拓扑 (topo)
- 测度 (measure)
- 度量 (metric) / 几何
- 等价关系
- 范畴 (category)
- 微分结构

计算机的诞生侧近了离散数学等难以结构化的数学分支, 因此现代不再按*结构*划分数学领域.
