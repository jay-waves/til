### 图像理解

二值图像有几何特征和拓扑特征, 在此基础上还有对其区域/边界/纹理的描述. 描述图像后, 才能对图像中目标进行决策分类(识别).

> [图像描述](图像描述.md)

一般流程为：图像平滑、边缘检测、特征提取、决策分类。
