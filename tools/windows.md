## Windows11 Defloat 

删除所有 AI 功能：

[zoicware/RemoveWindowsAI](https://github.com/zoicware/RemoveWindowsAI)

删除 Windows Defender （除防火墙外）：

[ionuttbara/windows-defender-remover](https://github.com/ionuttbara/windows-defender-remover)

Windows11 瘦身：

[Raphire/Win11Debloat](https://github.com/Raphire/Win11Debloat)

[ChrisTitusTech/winutil](https://github.com/ChrisTitusTech/winutil)

删除开始菜单：

[anymor/1Click-StartMenu](https://github.com/amymor/1Click-StartMenu)

删除 Windows 索引（用 Everything 替代即可）
* 禁用 Windows Search 服务
* 移除硬盘索引
* [禁用 Web SearchHost](https://github.com/shoober420/windows11-scripts/blob/main/DisableSearchHost.bat)

## 文件系统优化

[winfsp, FUSE for Windows](https://github.com/winfsp/winfsp)

[sshfs-win](https://github.com/winfsp/sshfs-win) 挂在远程 ssh 服务器硬盘

ExplorerPatcher 

SMB `\\server\share` 局域网文件共享，Windows 功能

VeryCrypt 硬盘加密

清理和定制右键菜单：ContextManager。推荐先关掉 Windows11 的折叠菜单风格

文件扩展名关联：FileTypesMan
