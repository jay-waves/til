品味如下:
- 命令行优先，GUI 不丑
- 可以复杂，但不能繁琐丑陋，不能绕圈子解决问题。
- 开源优先。
- 多平台，跨平台功能不过分阉割。
- 控制资源占用。不喜欢 Java。不喜欢 Chome 套壳，可以直接运行在浏览器 
- 功能不搞大杂烩

***
