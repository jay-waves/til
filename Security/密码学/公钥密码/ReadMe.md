公钥密码特征为:
- 生成密钥对的算法较容易
- 用公钥加密明文的算法较容易
- 用密钥解密密文的算法较容易
- 由公钥很难得到对应私钥
- 由密文和公钥很难得到对应的明文
- 公钥私钥可交换, 即 $K\cdot K^{-1}=K^{-1}\cdot K$

满足上述要求, 实质就找到了一个**单陷门函数** $f: X\to Y$. 

即 $\forall x\in X$, $f(x)$ 计算较容易; 而对几乎所有 $y\in Y$, $f^{-1}(y)$ 是计算上不可行的. 但是, 若掌握陷门 $z$, 则求 $f^{-1}(y)$ 较容易.
