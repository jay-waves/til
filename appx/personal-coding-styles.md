
- C 代码使用 Linux Kernel 风格
- Go 代码使用 Google 风格.
- C++ 代码使用标准库风格, 即全小写.
- Python 使用 pep8-style 风格.
- 不要求控制行长度, 建议代码行长度不超过 100 字符, 全角占两字符.

> Don't Repeat Yourself

1. 函数不要超过 20 行，注意代码可读性、简洁性。
2. 避免过深的缩进，简化控制语句嵌套
3. 变量名的长度，和变量名对程序的影响范围正相关
4. 按改动频率、代码热点来拆分代码结构和优化性能。不要一开始就过度设计。
5. 数据默认**不可变** (const)，函数默认是纯函数。
6. 移除全局共享的数据。
7. 变量初始化和变量声明写在一起。
