### 线性代数

矩阵:
```wolfram
{
	{1, 2}
	{3, 4}
}
```

```wolfram
Det[..] (* 计算矩阵的行列式 *)
Inverse[..] (* 计算矩阵的逆 *)
Eigenvalues[]
Eigenvactors[]

LinearSolve[] (* 线性方程组 *)
```

### 多项式代数

```wolfram

```
