品味如下:
- 界面不能太丑, 并且保持简洁和现代. (除非古早的神器...不可避免得丑)
- 开源, 免费, 有社区版优先. 不喜欢自行编译, 除非性能需要.
- 功能够用, 多平台 (起码支持 Windows + Debian)
- 不能占用太多资源. (不喜欢 web 应用和 java 应用)
- 使用方式可以复杂, 但不能繁琐丑陋.
