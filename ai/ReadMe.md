## 人工智能

机器学习 (Machine Learning) 有如下分支:
- 有监督学习: 用有标签的数据进行训练
- 无监督学习: 从混乱数据中提取特征
- 强化学习: 有外部奖励和反馈
- 深度学习 (Deep Learning): 模型使用神经网络.
