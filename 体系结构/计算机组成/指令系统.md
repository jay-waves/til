指令集结构 (ISA, Instruction Set Architecture) 可以分为两类: (尽管有融合趋势)

|                          | CISC (精简指令集) | RISC (复杂指令集) |
| ------------------------ | ----------------- | ----------------- |
| 指令系统                 | 复杂, 庞大        | 简单, 精简        |
| 指令数目, 格式, 寻址方式 | 多                | 较少              |
| 指令字长                 | 变长              | 等长              |
| 访存指令                 | 较多              | 仅 `LOAD/STORE`   |
| 指令执行时间             | 可能有多周期指令  | 大多是单周期指令  |
| 目标码体积               | 较短              | 较长              |
| 控制器                   | 微程序            | 主要为硬布线      |
| 例子                     | Intel             | ARM, PowerPC                  |

## 指令寻址方式

有四种程序分支类别:
- 顺序
- 有条件跳转
- 无条件跳转
- 函数调用 和 函数返回
